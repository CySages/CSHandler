CySages Handler Documentation

Installation

npm install cysages-handler@latest

Usage

const { CSHandler } = require('cysages-handler');

const config = {
    token: 'YOUR_BOT_TOKEN',
    mongoURL: 'YOUR_MONGO_DB_URL',
    logChannelId: 'YOUR_LOG_CHANNEL_ID',
    developers: ["DEV1", "DEV2"],
    prefix: '!'
}

const bot = new CSHandler(config);

bot.start();

Configuration

token: Your Discord bot token.
        mongoURL: MongoDB connection URL.
        commandsFolder: Path to the commands folder. (default: '../../Commands')
        intents: Array of Discord intents. (default: all intents)
        partials: Array of Discord partials. (default: all partials)
        activityName: Default activity name. (default: 'Default Activity')
        activityType: Default activity type. (default: 'PLAYING')

Important Note

Make sure to replace placeholder values like 'YOUR_BOT_TOKEN' and 'YOUR_MONGO_DB_URL' with the actual information for your bot. Customize the features, installation instructions, and other sections based on your bot's functionality.
